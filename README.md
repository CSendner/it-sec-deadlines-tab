# Introduction

Opens IT Security Deadlines in a Thunderbird tab. Once installed, you should see a button
for Security Deadlines. Clicking on it will open Security Deadlines Website within Thunderbird in a tab.

## Usage

- Click to load website/tab

# Version compatibility

- Tested with Thunderbird v148.
